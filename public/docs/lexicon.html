Unity desktop build hosted for download. To play, download and run the executable.

Download

Run Instructions (Windows)

  1. Download the EXE above.
  2. Unzip/extract if your browser saves it inside a folder.
  3. Double-click Lexicon.exe to play. If SmartScreen warns, choose “More info” → “Run anyway”.
