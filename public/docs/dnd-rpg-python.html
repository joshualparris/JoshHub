D&D RPG Dungeon (Python)

Text-based dungeon crawler located at Games/AAA/AAA. Run locally with Python; this page exposes the instructions from JoshHub.

How to Play Locally

  1. Open a terminal and navigate to the game folder:
    cd "/mnt/c/Users/joshu/OneDrive - LA TROBE UNIVERSITY/Desktop/Projects/Games/AAA/AAA"
  2. Run the game:
    python main.py
  3. Follow the prompts to create your character and descend through the dungeon.

Features

  • 4 classes, 4 races; dynamic combat, XP/leveling, gold and inventory.
  • 5-level dungeon with a final dragon boss.
  • No external dependencies beyond Python 3.10+.
