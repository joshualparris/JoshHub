Boundary Road App

Goal: turn captured 360° panorama images for 101 Boundary Rd into a navigable WSAD/Mouse 3D walkthrough.

Current Assets

  • Panorama URLs saved from ipropertyexpress viewer (9 shots: bathroom, hall, kitchen/living, patio, garage, etc.).
  • Design prompt drafted for a React + TypeScript + Three.js pano viewer with hotspots and floor plan.
  • Reference docs: Docs/BoundaryRoadApp.txt (this summary is pulled from there).

Plan Snapshot

  • Stack: React + TypeScript + Vite + Three.js (pointer-lock, inverted sphere pano rendering).
  • Controls: Mouse look + WSAD; hotspots to move between rooms; fade transitions.
  • Edit mode: add hotspots in-view, choose destinations, export updated house.json.
  • Optional floor plan: SVG nodes/edges, click to jump to rooms, drag to reposition in edit mode.
