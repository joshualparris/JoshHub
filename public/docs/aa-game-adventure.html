Repository Layout

  • src/engine: pure deterministic game logic (state, step, map, schema, commands, systems).
  • src/adapters: loop, canvas, input, audio, persistence, renderer.
  • src/mods: optional plugins (combat, shop, procedural, etc.).
  • tools/: helpers for applying example maps.
